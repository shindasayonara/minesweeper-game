Minesweeper

    Replay Game

      Справка

      Доступные функции:
      - Новая игра: Позволяет начать новую игру с заданными параметрами (ширина, высота и количество мин).
      - Просмотр прошлых игр: Открывает список всех сохранённых игр. Чтобы посмотреть реплей конкретной игры, нажмите на кнопку "View Past Games" и выберите интересующую игру из списка.
      - Повтор игры (Replay): Показывает пошаговую историю выбранной игры, включая информацию о каждом ходе (координаты и результат).
      - Очистка базы данных: Удаляет все сохранённые игры из базы данных.

      Как играть:
      1. Введите ваше имя в поле "Player Name".
      2. Установите параметры игры (ширина, высота и количество мин).
      3. Нажмите кнопку "New Game", чтобы начать новую игру.
      4. Щёлкайте по ячейкам на игровом поле, чтобы открыть их. Если вы откроете ячейку с миной, игра закончится.
      5. Цель игры — открыть все ячейки без мин.

      Реплей игры:
      Чтобы просмотреть повтор игры:
      1. Нажмите кнопку "View Past Games".
      2. В появившемся списке выберите игру, кликнув на неё.
      3. Реплей игры будет отображён в отдельном окне, где вы сможете увидеть все шаги и результат.
